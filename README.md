## About Me

I'm a **Full-Stack Software Developer** specializing in **Node.js backend engineering** and **React-based frontend development**. I build production-grade applications focused on performance, scalability, and clean architecture.

My experience spans RESTful API design, database modeling (MongoDB & PostgreSQL), and cloud deployment on **AWS**. I'm passionate about writing maintainable code, applying solid engineering principles, and continuously leveling up through real-world projects.
